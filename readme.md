## weapp-template-dev

[![Greenkeeper badge](https://badges.greenkeeper.io/zanjs/weapp-template-dev.svg)](https://greenkeeper.io/)

微信小程序构建工程模板

### install

```ssh
git clone https://github.com/zanjs/weapp-template-dev.git

cd weapp-template-dev

yarn
```

构建开发开发

```
npm start
```

编译生产环境开发

```
npm run build
```
